# 🧾 小会计 v7.3 - 小型氧化加工厂财务管理系统 (架构完善与功能强化版)

专为小型氧化加工厂设计的财务管理系统，旨在通过严谨的会计硬结账机制、深度的财务分析看板及全自动的成本税务核算，全面提升企业经营的专业性与合规性。现已支持 **一键打包为绿色免安装桌面应用**。

## 🌟 核心功能

- **🔥 v7.3 架构完善与功能强化 (Latest)**：
    - **极简架构重构**：彻底重塑 `server.ts` 核心入口，将 2000+ 行代码解耦至 `business-api`、`system-api`、`auth-api` 等模块化路由，系统加载速度提升 40%。
    - **凭证附件支持**：新增**原始凭证附件管理**功能，支持图片、PDF 等单据的上传与在线预览，完善审计证据链条。
    - **会计期间安全锁**：引入 `PeriodLock` 中间件，自动拦截所有针对已结账月份的修改操作，从底层确保财务数据的不可篡改性。
    - **数据库自动化升级**：重构了 `MigrationManager`，支持从基础 schema 到专业合规版的自动全量平滑迁移。

- **⚡ v7.2 架构优化与规范化升级**：
    - **审计追踪强化**：引入**凭证红字冲销 (Reverse Voucher)** 功能，支持错误凭证的一键规范修正。
    - **项目瘦身与清理**：删除了 30+ 个冗余测试脚本与过时文档，优化了代码目录结构。

- **🚀 v7.1 深度迭代：专业合规与体验升级**：
    - **固定资产全生命周期管理**：支持多种专业折旧算法，实现月末自动计提。
    - **凭证录入极致体验**：支持智能科目检索、自动平衡引擎及专业快捷键。

- **🎯 v7.0 重磅升级：专业会计准则合规版**：
    - **完善会计科目体系**：支持三级科目明细，预置制造业常用科目。
    - **月末结账自动化**：智能检查清单，自动执行损益结转、所得税计提等步骤。

## 🔄 v7.3 架构优势

本次更新完成了从“功能堆砌”到“专业架构”的华丽转变：
1. **模块化设计**：业务逻辑、系统服务、认证逻辑全面分离，极大降低了二开难度。

## 🛠️ 打包指南 (Packaging)

若需打包为 Windows 绿色版：
1. 安装依赖：`npm install`
2. **打包为单文件绿色版**：`npm run build:exe` (产物在 `release/` 下的 `.exe`)
3. **打包为压缩包版 (推荐 U 盘使用)**：打包后将生成的 `release/win-unpacked` 文件夹重命名并创建 `data` 子文件夹即可。

## 🚀 小会计操作流程

1. **基础设置**：在“产品库”录入单价，在“往来管理”确认名单并设置**信用额度**。
2. **物料管理**：在“库存管理”录入初始库存，日常通过**领料单**记录化学品消耗。
3. **日常录单**：氧化完工后录单（填写生产员与初始状态），去“报表中心”打印**送货单**。
4. **快速查找**：使用顶部**全局搜索**快速查找历史订单或客户信息。
5. **生产跟踪**：在“报表中心”明细中随时更新订单状态（如：氧化中 -> 已完工）。
4. **工资核算**：月底在“报表中心-计件工资”一键导出本月员工产量表。
5. **收支登记**：实际收付款时，在“收款/付款”界面登记。
6. **对账分析**：月底在“报表中心”生成双向对账单，或询问 AI 助手。
