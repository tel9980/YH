# 🧾 小会计 v5.8 - 小型氧化加工厂财务管理系统

专为小型氧化加工厂设计的财务管理系统，旨在帮助“会计小白”高效处理繁杂的加工费核算、收支管理及客户对账。

## 🌟 核心功能

- **🇨🇳 国内会计深度优化 (v5.8 重磅更新)**：
    - **税务合规核算 (Tax Compliance)**：引入税务工作台，自动计算应交增值税（销项减进项）并预估企业所得税。支持根据订单发票状态自动勾稽，辅助会计进行月度纳税申报。
    - **利润分配管理 (Profit Appropriation)**：支持计提法定盈余公积、任意盈余公积及向投资者分配分红。自动追踪未分配利润余额，规范工厂净资产管理。
    - **杜邦财务分析 (DuPont Analysis)**：在健康看板引入杜邦分析模型，将 ROE 拆解为销售净利率、资产周转率与权益乘数，穿透式分析工厂盈利根源。
    - **手工记账凭证 (v5.7)**：支持会计进行年终调整、税费计提等专业手工调账。
    - **财务健康看板 (v5.7)**：通过雷达图与核心财务比率，实时掌握工厂经营风险。
- **📊 智能仪表盘 (v4.8)**：
    - **生产员龙虎榜 (v3.9)**：实时统计产量前 5 名的员工，激发生产积极性。
    - **财务健康体检 (v3.8)**：自动扫描异常库存、零价订单及长期欠款。仪表盘新增“信用风险预警”组件。实时监控所有客户的欠款情况，一旦超过预设的“信用额度”，系统将自动发出红色预警，防范坏账风险。
    - **库存价值核算 (v4.0)**：仪表盘新增“库存价值”核心指标。系统根据原材料的实时库存与成本单价，自动计算当前仓库的总资产价值，帮助老板掌握资金占用情况。
    - **生产员龙虎榜 (v3.9)**：新增“生产员龙虎榜”组件。实时统计并展示产量前 5 名的员工，激发生产积极性，同时方便会计快速了解车间产能分布。
    - **财务健康体检 (v3.8)**：新增“健康体检”组件。自动扫描系统中的**异常库存**（负数）、**零价订单**（待核价）及**长期欠款**（超 90 天），并提供 AI 优化建议，帮助会计防范风险。
    - **支出构成分析 (v3.8)**：仪表盘新增支出占比饼图，直观展示各项费用（如房租、水电、材料）的分布情况。
    - **收支趋势图**：新增近 6 个月收支趋势对比图（收入、支出、产值），财务状况一目了然。
    - **环比分析 (MoM)**：核心财务指标（收入、支出、订单、利润）新增月度环比增长/下降百分比。
    - **现金流预测**：基于历史数据，自动预测未来 30 天的现金流走势（收入、支出、净流向）。支持**一键复制预测报告**。
    - **骨架屏加载**：优化首屏加载体验，减少白屏等待感。
- **📋 增强版报表中心 (v4.0)**：
    - **资产概览增强 (v4.0)**：报表中心“财务概览”新增库存总价值统计，实现收入、支出、利润与资产价值的同屏展示，财务报表更完整。
    - **应付账款账龄分析 (v3.9)**：在“账龄分析”页签中新增“应付账款”维度。基于供应商账单与实际付款记录，自动测算对供应商的欠款时长（0-30/31-60/61-90/90+天），辅助工厂合理安排资金支付计划。
    - **对账单核对标记 (v3.8)**：客户对账单新增“核对”功能。会计可以对每一笔订单进行勾选标记，方便记录哪些项目已与客户确认，哪些尚在争议中，对账过程更清晰。
    - **支出深度分析 (v3.8)**：报表中心新增“支出分析”页签。支持按分类统计支出总额及占比，辅助工厂进行成本管控。
    - **全局搜索 (v3.7)**：顶部导航新增全局搜索框。支持一键跨模块搜索订单、客户、产品，快速定位目标数据。
    - **客户信用额度 (v3.7)**：在“往来管理”中可为客户设置信用额度（欠款上限）。当客户欠款接近或超过额度时，系统将提供视觉提醒，辅助会计把控回款风险。
    - **物料领用管理 (v3.7)**：库存管理新增“领料单”功能。支持记录生产员领用化学品（如三酸、片碱）的日期、数量及用途，系统自动扣减库存并生成流水，实现精细化成本控制。
    - **模块自定义管理 (v3.6)**：新增“模块管理”功能。用户可以根据工厂实际业务，自由开启或关闭不需要的功能模块，并支持对模块进行重命名（如将“录单”改为“送货单录入”），让系统更贴合个性化需求。
    - **计件工资报表 (v3.6)**：新增“计件工资”页签。自动根据录单时填写的“生产员”统计周期内的总产量、订单笔数及平均产量，支持**一键导出工资表**，方便会计核算人工成本。
    - **生产状态追踪 (v3.6)**：订单新增“生产状态”字段（待产、氧化中、待检、已完工、已送货）。支持在报表中直接下拉更新状态，实时掌握车间进度。
    - **批量操作 (v3.5)**：支持订单的**批量勾选**，可一键完成“批量标记已开票”、“批量标记未开票”或“批量删除”，极大提升对账效率。
    - **正式对账单打印 (v3.5)**：新增“正式对账单”预览与打印功能。包含工厂抬头、往来明细、合计金额及**模拟公章**，支持直接打印为纸质版或另存为 PDF 发送给客户。
    - **开票管理 (v3.4)**：支持在报表中一键标记“已开票/未开票”，并提供开票状态筛选。
    - **利润分析 (v3.3)**：通过图表展示产品利润贡献 Top 10 及客户产值占比。
    - **审计日志**：支持查看系统操作日志，确保每一笔修改都有迹可循。
- **📋 往来管理 (v4.1)**：
    - **信用额度设置 (v4.1)**：支持为每个客户设置独立的“信用额度”。
    - **录单实时提醒 (v4.1)**：在“批量录单”界面，选择客户后系统会自动显示其当前欠款与信用额度，若已超额则高亮提醒，辅助决策是否继续接单。
    - **往来单位导入**：支持客户与供应商的**快速添加**及 **Excel 批量导入**。
- **📦 智能产品库 (v3.4)**：
    - **税率计算器 (v3.4)**：录单时支持选择不同税率（0%、3%、6%、13%），内置“含税/不含税”转换工具。
    - **批量调价**：支持产品单价的批量更新。
    - **Excel 导入**：预设常用氧化产品、单位（件、条、只、个、米长、米重、公斤、平方米）及单价，支持 Excel 批量导入。
- **📱 移动端与 PWA (v2.9)**：响应式布局适配手机，支持**离线访问与安装 (PWA)**。
- **📥 数据导入与备份 (v3.0)**：首页新增**“一键备份”**，可将全系统数据导出为 JSON 文件。支持产品库和初始库存的 Excel 批量导入。
- **📈 账龄分析 (Aging Report)**：报表中心新增**应收账款账龄分析图表**。基于先进先出 (FIFO) 算法，自动识别逾期 30/60/90 天以上的欠款。
- **📜 库存流水账 (Ledger)**：库存管理新增**变动历史**。记录每一次入库、出库、盘点的详细时间、数量及备注，实现物料流转全追溯。
- **📋 灵活录单**：支持多种计价方式，支持**“批量录单”模式**，支持**“挂具费”**独立核算及**“附件/图片”**备注。
- **💰 收支与账务**：
    - **应收/收款**：记录客户订单及实际回款，自动核算客户欠款。

- **📦 精细化库存管理 (v4.0)**：
    - **成本单价管理 (v4.0)**：库存项目新增“成本单价”字段。支持在录入或导入库存时设置单价，系统自动计算每项物料的库存价值。
    - **领料单管理 (v3.7)**：支持记录生产员领用化学品（如三酸、片碱）的日期、数量及用途，系统自动扣减库存并生成流水。

- **📊 报表中心**：一键生成利润表、支出分布。支持 **Excel 导出**对账单，支持**送货单打印**。
- **🔒 月结归档**：支持按月锁定财务数据，归档后历史账目不可修改。
- **📜 审计日志**：自动记录订单增删改操作，确保财务数据安全可追溯。
- **🤖 AI 智能助手**：接入 Gemini AI，支持**快捷提问**（如：谁欠款最多、哪些库存不足），财务状况秒掌握。

## 🚀 小会计操作流程

1. **基础设置**：在“产品库”录入单价，在“往来管理”确认名单并设置**信用额度**。
2. **物料管理**：在“库存管理”录入初始库存，日常通过**领料单**记录化学品消耗。
3. **日常录单**：氧化完工后录单（填写生产员与初始状态），去“报表中心”打印**送货单**。
4. **快速查找**：使用顶部**全局搜索**快速查找历史订单或客户信息。
5. **生产跟踪**：在“报表中心”明细中随时更新订单状态（如：氧化中 -> 已完工）。
4. **工资核算**：月底在“报表中心-计件工资”一键导出本月员工产量表。
5. **收支登记**：实际收付款时，在“收款/付款”界面登记。
6. **对账分析**：月底在“报表中心”生成双向对账单，或询问 AI 助手。
