## 系统概述

小会计 v7.0 是专为小型氧化加工厂设计的专业财务管理系统，符合《企业会计准则》和《小企业会计准则》要求。本版本引入了完善的会计科目体系、凭证自动生成引擎、标准成本核算、月末结账自动化等专业功能。

### v7.0 核心升级

- **会计准则合规**：三级科目体系，预置制造业科目
- **业务财务一体化**：业务单据自动生成会计凭证
- **成本核算精细化**：标准成本法与实际成本法，成本差异分析
- **财务分析专业化**：符合准则的三大报表，财务指标分析
- **税务管理智能化**：增值税和企业所得税自动计算与申报

## 快速开始

### 首次启动

1. **数据库自动升级**：首次启动 v7.0 时，系统会自动执行数据库迁移，将 v6.5 数据升级到 v7.0。迁移过程约需 10-30 秒，请勿关闭程序。

2. **检查迁移结果**：迁移完成后，系统会显示迁移日志。请检查是否有错误提示。

3. **备份数据**：建议在升级前备份 `xiaokuaiji.db` 文件，以防万一。

### 基础配置

1. **会计科目设置**：进入"设置 → 会计科目"，查看预置的制造业科目。如需新增科目，请遵循三级科目编码规则（如 6602-01-001）。

2. **税务配置**：进入"税务管理 → 税务配置"，设置纳税人类型（一般纳税人/小规模纳税人）和税率。

3. **库存计价方法**：进入"设置 → 库存管理"，选择库存计价方法（先进先出法/加权平均法/移动加权平均法）。

4. **产品成本设置**：进入"成本管理 → 产品成本"，为每个产品设置标准成本（材料、人工、制造费用）。

## 核心功能使用指南

### 1. 会计科目管理

#### 查看科目树

进入"设置 → 会计科目"，系统以树形结构展示所有科目。科目按类型分类：资产、负债、权益、收入、成本、费用。

#### 新增科目

1. 点击"新增科目"按钮
2. 填写科目信息：
   - **科目编码**：遵循三级编码规则（一级：4位，二级：+2位，三级：+3位）
   - **科目名称**：如"直接材料"
   - **科目类型**：选择资产/负债/权益/收入/成本/费用
   - **父科目**：选择上级科目（如"生产成本"）
   - **辅助核算**：勾选需要的辅助核算类型（客户/供应商/部门/项目/存货）
3. 点击"保存"

#### 科目状态管理

- **启用/停用**：点击科目右侧的状态开关，可切换科目的启用状态。停用的科目不可用于新凭证。
- **删除保护**：有发生额的科目不可删除，系统会提示错误。

### 2. 凭证管理

#### 手工录入凭证

1. 进入"凭证管理 → 新增凭证"
2. 填写凭证信息：
   - **凭证日期**：选择记账日期
   - **凭证类型**：手工凭证/销售凭证/采购凭证等
   - **摘要**：简要说明业务内容
3. 添加凭证分录：
   - 选择科目（支持智能搜索：科目编码/名称/拼音首字母）
   - 填写借方或贷方金额
   - 如科目设置了辅助核算，需填写辅助信息（客户/供应商/部门等）
4. 系统自动验证借贷平衡
5. 点击"保存"

#### 凭证自动生成

系统支持从业务单据自动生成凭证：

- **销售订单**：完成订单后，自动生成销售收入凭证和成本结转凭证
- **收款单**：录入收款后，自动生成收款凭证（借：银行存款/库存现金，贷：应收账款）
- **采购单**：采购入库后，自动生成采购凭证（借：原材料/库存商品，贷：应付账款）
- **费用支出**：录入费用后，自动生成费用凭证

**操作步骤**：
1. 进入"凭证管理 → 批量生成凭证"
2. 选择业务单据类型和期间
3. 勾选需要生成凭证的单据
4. 点击"批量生成"
5. 系统自动生成凭证并显示结果

#### 凭证审核

1. 进入"凭证管理"，查看凭证列表
2. 点击凭证右侧的"审核"按钮
3. 审核后的凭证不可修改或删除（需先取消审核）

#### 凭证模板

**保存模板**：
1. 录入常用凭证后，点击"保存为模板"
2. 填写模板名称和描述
3. 点击"保存"

**使用模板**：
1. 新增凭证时，点击"从模板创建"
2. 选择模板
3. 系统自动填充凭证分录，可根据实际情况修改金额

### 3. 库存计价管理

#### 设置计价方法

1. 进入"设置 → 库存管理"
2. 选择计价方法：
   - **先进先出法（FIFO）**：按最早入库批次计算出库成本
   - **加权平均法**：库存总金额 ÷ 库存总数量
   - **移动加权平均法**：每次入库更新平均单价
   - **个别计价法**：按实际批次计价
3. 点击"保存"

**注意**：计价方法变更会影响后续出库成本计算，请谨慎操作。

#### 查看库存变动记录

1. 进入"库存管理 → 存货明细账"
2. 选择库存项目
3. 查看每次入库、出库的数量、单价、金额和余额

#### 库存成本调整

1. 进入"库存管理 → 成本调整"
2. 选择库存项目
3. 填写调整原因（盘盈/盘亏/价格调整）
4. 填写调整金额
5. 系统自动生成库存调整凭证

### 4. 成本核算管理

#### 设置产品标准成本

1. 进入"成本管理 → 产品成本"
2. 选择产品
3. 填写标准成本：
   - **标准材料成本**：单位产品的材料成本
   - **标准人工成本**：单位产品的人工成本
   - **标准制造费用**：单位产品的制造费用
4. 系统自动计算标准总成本
5. 点击"保存"

#### 生产订单管理

1. 进入"成本管理 → 生产订单"
2. 点击"新增生产订单"
3. 填写订单信息：
   - **订单编号**：如 PO202401001
   - **产品**：选择生产产品
   - **数量**：生产数量
   - **开工日期**：生产开始日期
4. 点击"保存"

#### 成本归集

- **材料成本**：领用原材料时，系统自动归集到对应生产订单
- **人工成本**：录入计件工资时，系统自动归集到对应生产订单
- **制造费用**：月末结账时，系统自动分配制造费用到各生产订单

#### 成本差异分析

1. 进入"成本管理 → 成本差异"
2. 选择会计期间
3. 查看成本差异报表：
   - **材料价格差异**：实际材料单价与标准单价的差异
   - **材料数量差异**：实际材料用量与标准用量的差异
   - **人工效率差异**：实际人工成本与标准人工成本的差异
   - **制造费用差异**：实际制造费用与标准制造费用的差异
4. 差异率超过阈值（如 ±10%）时，系统会在仪表盘显示预警

#### 成本差异处理

1. 在成本差异报表中，点击"处理差异"
2. 选择处理方式：
   - **结转至主营业务成本**：全部差异计入当期损益
   - **在产品/库存商品间分摊**：按比例分摊差异
3. 系统自动生成成本差异结转凭证

### 5. 月末结账

#### 启动结账流程

1. 进入"结账管理 → 月末结账"
2. 选择结账期间（如 2024-01）
3. 点击"启动结账"

#### 结账前置条件检查

系统会自动检查以下条件：
- ✓ 所有业务单据已生成凭证
- ✓ 所有凭证借贷平衡
- ✓ 银行存款已完成对账
- ✓ 固定资产已计提折旧
- ✓ 成本已完成归集和分配

如有未满足的条件，系统会列出待处理事项，阻止结账。

#### 结账步骤

系统按以下顺序自动执行：
1. **计提固定资产折旧**：为所有在用固定资产计提折旧
2. **分配制造费用**：按设定的分配方法分配制造费用到各生产订单
3. **结转生产成本**：将完工产品成本从生产成本结转到库存商品
4. **处理成本差异**：结转成本差异到主营业务成本
5. **结转损益**：将收入、成本、费用结转到本年利润
6. **计提所得税**：按利润总额计提企业所得税
7. **锁定期间**：锁定该会计期间，禁止修改

#### 查看结账报告

结账完成后，系统生成结账报告，包括：
- 本月关键财务指标（收入、成本、费用、净利润）
- 财务比率（毛利率、净利率、费用率）
- 异常提示和警告

#### 反结账

如需修改已结账月份的数据：
1. 进入"结账管理 → 月末结账"
2. 选择已结账期间
3. 点击"反结账"
4. 系统记录反结账操作的审计日志
5. 期间解锁，可修改数据

**注意**：反结账操作会被记录在审计日志中，请谨慎操作。

### 6. 财务报表

#### 资产负债表

1. 进入"报表中心 → 资产负债表"
2. 选择报表期间（月报/季报/年报）
3. 查看报表：
   - **资产**：流动资产、非流动资产
   - **负债**：流动负债、非流动负债
   - **所有者权益**：实收资本、留存收益、本年利润
4. 支持期间对比（本期与上期、本期与去年同期）

#### 利润表

1. 进入"报表中心 → 利润表"
2. 选择报表期间
3. 查看报表：
   - **营业收入**：主营业务收入、其他业务收入
   - **营业成本**：主营业务成本、其他业务成本
   - **期间费用**：销售费用、管理费用、财务费用
   - **利润**：营业利润、利润总额、净利润

#### 现金流量表

1. 进入"报表中心 → 现金流量表"
2. 选择报表期间和编制方法（直接法/间接法）
3. 查看报表：
   - **经营活动现金流**：销售商品、购买商品、支付工资等
   - **投资活动现金流**：购建固定资产、处置固定资产等
   - **筹资活动现金流**：吸收投资、分配股利等

#### 财务指标分析

1. 进入"报表中心 → 财务指标"
2. 查看财务指标：
   - **偿债能力**：流动比率、速动比率、资产负债率
   - **营运能力**：应收账款周转率、存货周转率、总资产周转率
   - **盈利能力**：销售毛利率、销售净利率、净资产收益率、总资产收益率

#### 报表导出

1. 在报表页面，点击"导出"按钮
2. 选择导出格式（Excel/PDF）
3. 系统生成文件并下载

#### 报表打印

1. 在报表页面，点击"打印预览"按钮
2. 设置报表抬头和公章
3. 点击"打印"

### 7. 辅助核算

#### 客户往来明细账

1. 进入"辅助核算 → 客户往来"
2. 选择客户和期间
3. 查看明细账：
   - 期初余额
   - 本期发生额（借方/贷方）
   - 期末余额

#### 供应商往来明细账

1. 进入"辅助核算 → 供应商往来"
2. 选择供应商和期间
3. 查看明细账

#### 往来对账单

1. 进入"辅助核算 → 往来对账"
2. 选择客户/供应商和期间
3. 查看对账单：
   - 销售订单/采购单明细
   - 收款/付款记录
   - 自动匹配核销
   - 期末余额
4. 支持导出和打印

### 8. 税务管理

#### 税务配置

1. 进入"税务管理 → 税务配置"
2. 设置纳税人类型：
   - **一般纳税人**：增值税税率 13%
   - **小规模纳税人**：增值税征收率 3%
3. 设置企业所得税税率（通常 25%）
4. 点击"保存"

#### 增值税申报表

1. 进入"税务管理 → 增值税申报"
2. 选择申报期间（月度/季度）
3. 查看申报表：
   - **销项税额**：销售收入对应的增值税
   - **进项税额**：采购支出对应的增值税
   - **应纳税额**：销项税额 - 进项税额
4. 支持导出为税务局电子申报格式

#### 企业所得税申报表

1. 进入"税务管理 → 企业所得税申报"
2. 选择申报期间（季度）
3. 查看申报表：
   - **营业收入**
   - **营业成本**
   - **期间费用**
   - **应纳税所得额**
   - **应纳所得税额**：应纳税所得额 × 25%
4. 支持导出

#### 税务风险提示

1. 进入"税务管理 → 税务风险"
2. 查看风险提示：
   - 进项税额占比异常
   - 税负率偏低
   - 长期零申报
3. 根据提示采取相应措施

### 9. 固定资产管理

#### 新增固定资产

1. 进入"固定资产管理 → 资产列表"
2. 点击"新增资产"
3. 填写资产信息：
   - **资产编号**：如 FA202401001
   - **资产名称**：如"氧化生产线"
   - **资产类别**：机器设备/运输工具/电子设备等
   - **购入日期**
   - **原值**：购入价格
   - **预计使用年限**：如 10 年
   - **预计净残值**：如原值的 5%
   - **折旧方法**：年限平均法/双倍余额递减法/年数总和法/工作量法
4. 点击"保存"

#### 查看折旧计划

1. 在资产列表中，点击资产右侧的"折旧计划"
2. 查看每月折旧额和累计折旧

#### 月末自动计提折旧

月末结账时，系统自动为所有在用固定资产计提折旧，并生成折旧凭证：
- 借：制造费用/管理费用
- 贷：累计折旧

#### 固定资产处置

1. 在资产列表中，点击资产右侧的"处置"
2. 填写处置信息：
   - **处置日期**
   - **处置收入**：出售价格
   - **处置费用**：清理费用
3. 系统自动计算处置损益：处置收入 - 账面净值 - 处置费用
4. 系统自动生成处置凭证

### 10. 数据校验与合规检查

#### 凭证验证

系统在保存凭证时自动验证：
- ✓ 借贷平衡
- ✓ 科目借贷方向合理性
- ✓ 金额精度（最多两位小数）
- ✓ 会计期间是否已结账
- ✓ 辅助核算必填项

#### 异常余额检查

系统自动检查以下异常情况并提示警告：
- ⚠ 银行存款余额为负数
- ⚠ 库存数量为负数
- ⚠ 应收账款贷方余额（预收款项）
- ⚠ 应付账款借方余额（预付款项）

#### 数据一致性检查

1. 进入"设置 → 数据一致性检查"
2. 点击"执行检查"
3. 系统验证：
   - 总账与明细账一致性
   - 账表与单据一致性
   - 凭证与业务单据关联完整性
4. 查看检查结果和异常提示

## 常见问题解答

### Q1: 数据库升级失败怎么办？

**A**: 如果数据库升级失败，请按以下步骤操作：
1. 关闭程序
2. 恢复备份的 `xiaokuaiji.db` 文件
3. 重新启动程序
4. 如果问题仍然存在，请联系技术支持

### Q2: 如何修改已审核的凭证？

**A**: 已审核的凭证不可直接修改。需要先取消审核：
1. 进入"凭证管理"
2. 找到需要修改的凭证
3. 点击"取消审核"
4. 修改凭证
5. 重新审核

### Q3: 如何修改已结账月份的数据？

**A**: 已结账月份的数据不可直接修改。需要先反结账：
1. 进入"结账管理 → 月末结账"
2. 选择已结账期间
3. 点击"反结账"
4. 修改数据
5. 重新结账

**注意**：反结账操作会被记录在审计日志中。

### Q4: 库存计价方法可以随时更改吗？

**A**: 库存计价方法可以更改，但会影响后续出库成本计算。更改时系统会提示确认，并记录变更日志。建议在会计年度开始时更改，避免影响当期成本核算。

### Q5: 成本差异率多少算正常？

**A**: 一般认为成本差异率在 ±5% 以内属于正常范围。系统默认阈值为 ±10%，超过此阈值会触发预警。您可以根据企业实际情况调整阈值。

### Q6: 如何设置凭证模板？

**A**: 
1. 手工录入一张常用凭证
2. 点击"保存为模板"
3. 填写模板名称和描述
4. 下次录入相同类型凭证时，点击"从模板创建"即可

### Q7: 报表中显示"包含未审核数据"是什么意思？

**A**: 这表示报表期间内有未审核的凭证。未审核凭证的数据可能不准确，建议先审核所有凭证后再查看报表。

### Q8: 如何导出财务报表？

**A**: 
1. 在报表页面，点击"导出"按钮
2. 选择导出格式（Excel/PDF）
3. 系统生成文件并自动下载

### Q9: 固定资产折旧方法可以更改吗？

**A**: 固定资产折旧方法一旦确定，原则上不应更改。如确需更改，需在资产详情页面修改，系统会重新计算折旧计划。

### Q10: 如何备份数据？

**A**: 
1. 进入"设置 → 数据备份"
2. 点击"立即备份"
3. 系统自动备份数据库文件到指定目录
4. 建议定期备份，或在关键操作前手动备份

### Q11: 辅助核算是什么？如何使用？

**A**: 辅助核算是指在会计科目的基础上，增加客户、供应商、部门、项目、存货等维度的核算。使用方法：
1. 在科目设置中，为需要辅助核算的科目勾选辅助核算类型
2. 录入凭证时，系统会要求填写对应的辅助信息
3. 在"辅助核算"模块中，可按维度查询明细账

### Q12: 税务申报表数据不准确怎么办？

**A**: 请检查以下几点：
1. 税务配置是否正确（纳税人类型、税率）
2. 销售订单和采购单是否正确设置了税率
3. 是否有未生成凭证的业务单据
4. 凭证中的税费科目是否正确

### Q13: 如何查看操作日志？

**A**: 
1. 进入"设置 → 审计日志"
2. 选择查询期间和操作类型
3. 查看所有关键操作的详细记录（操作时间、操作人、操作内容）

### Q14: 系统支持多用户吗？

**A**: 当前版本为单机版，不支持多用户同时操作。如需多用户功能，请联系技术支持了解企业版。

### Q15: 如何联系技术支持？

**A**: 如遇到问题，请通过以下方式联系技术支持：
- 在系统中点击"帮助 → 反馈问题"
- 发送邮件至技术支持邮箱
- 查看项目 GitHub 仓库的 Issues
